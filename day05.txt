# Session 1 – C Programming (100 Minutes)

## Topic

### One-Dimensional Arrays

### Learn (30 Minutes)

Study these concepts:

* What is an array?
* Why use arrays?
* Declaring an array
* Initializing an array
* Accessing array elements
* Taking array input using a loop
* Printing array elements
* Array indexing (starts from 0)

Example:

```c
#include<stdio.h>

int main()
{
    int marks[5];
    int i;

    printf("Enter 5 marks:\n");

    for(i = 0; i < 5; i++)
    {
        scanf("%d", &marks[i]);
    }

    printf("Marks are:\n");

    for(i = 0; i < 5; i++)
    {
        printf("%d ", marks[i]);
    }

    return 0;
}
```

```c
#include<stdio.h>

{

    for(i = 1; i <= 5; i++)
    {
        scanf("%d", &a[i]);

    {
        printf("%d ", a[i]);

}
```

### Answer these:

# Session 2 – Aptitude (40 Minutes)

## Topic

### Average

### Learn (15 Minutes)

Formula:

**Average = Sum of all values ÷ Number of values**

Understand:

* Finding missing values
* New average after adding/removing a value

### Practice (25 Minutes)
