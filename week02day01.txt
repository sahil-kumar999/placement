# Session 1 – C Programming (100 Minutes)

## Topic

### Strings in C

### Learn (30 Minutes)

Study these concepts:

* What is a string?
* Character array vs string
* Null character (`'\0'`)
* Reading strings using `scanf()` and `fgets()`
* Printing strings
* String traversal using loops

Example:

```c
#include <stdio.h>

int main()
{
    char name[50];

    printf("Enter your name: ");
    fgets(name, sizeof(name), stdin);

    printf("Hello, %s", name);

    return 0;
}
```

## Debugging Challenge (20 Minutes)

Find and fix the errors:

```c
#include <stdio.h>

int main()
{
    char name;

    printf("Enter your name: ");
    scanf("%s", &name);

    printf("%s", name);

    return 0;
}
```

Answer:

* Why is `char name;` incorrect?
* Why is `%s` causing problems here?
* Rewrite the program correctly.

# Session 2 – Aptitude (40 Minutes)

## Topic

### Time, Speed and Distance

### Learn (15 Minutes)

Study:

* Speed = Distance / Time
* Distance = Speed × Time
* Time = Distance / Speed
* Unit conversions:

  * 1 km = 1000 m
  * 1 hour = 60 minutes
  * km/h ↔ m/s
