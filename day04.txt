# Session 1 – C Programming (95 Minutes)

## Topic

### Functions in C

### Learn (25 Minutes)

Study these concepts:

* What is a function?
* Advantages of functions
* Function declaration (prototype)
* Function definition
* Function call
* Parameters vs Arguments
* Return type (`void` vs `int`)

Example:

```c
#include<stdio.h>

int add(int a, int b);

int main()
{
    int result = add(10, 20);
    printf("Sum = %d", result);
    return 0;
}

int add(int a, int b)
{
    return a + b;
}
```

## Debugging Challenge (20 Minutes)

Find and fix the errors:

```c
#include<stdio.h>

void square(int n)
{
    return n*n;
}

int main()
{
    int ans = square(5);
    printf("%d", ans);
    return 0;
}
```

Answer:

* Why is this incorrect?
* Rewrite it correctly.
