## Debugging Challenge (20 Minutes)

Find and fix the errors:

```c
#include <stdio.h>

int main()
{
    int a[5] = {5, 2, 4, 1, 3};
    int i, j, temp;

    for(i = 0; i < 5; i++)
    {
        for(j = 0; j < 5; j++)
        {
            if(a[j] > a[j + 1])
            {
                temp = a[j];
                a[j] = a[j + 1];
                a[j + 1] = temp;
            }
        }
    }

    for(i = 0; i < 5; i++)
        printf("%d ", a[i]);

    return 0;
}
```

### Questions

* What causes the array out-of-bounds error?
* How should the inner loop condition be corrected?
* Rewrite the corrected program.

# Session 2 – Aptitude (40 Minutes)

## Topic

### Profit and Loss

### Learn (15 Minutes)

Study:

* Cost Price (CP)
* Selling Price (SP)
* Profit
* Loss
* Profit %
* Loss %

Formulas:

```text
Profit = SP − CP

Loss = CP − SP

Profit % = (Profit / CP) × 100

Loss % = (Loss / CP) × 100
```

### Practice (25 Minutes)

1. CP = ₹400, SP = ₹500. Find the profit and profit percentage.
2. CP = ₹850, SP = ₹765. Find the loss and loss percentage.
3. A shopkeeper gains 20% on an item costing ₹600. Find the selling price.
4. A bicycle is sold for ₹2,700 at a loss of 10%. Find the cost price.
5. A trader buys 15 notebooks for ₹450 and sells each for ₹35. Find the total profit or loss.
