# Session 1 – C Programming (100 Minutes)

## Topic

### String Library Functions (`string.h`)

### Learn (30 Minutes)

Study the following functions:

* `strlen()` – Find string length
* `strcpy()` – Copy one string to another
* `strcat()` – Concatenate (join) two strings
* `strcmp()` – Compare two strings

Example:

```c
#include <stdio.h>
#include <string.h>

int main()
{
    char str1[50] = "Hello";
    char str2[50] = " World";

    printf("Length: %lu\n", strlen(str1));

    strcat(str1, str2);

    printf("%s", str1);

    return 0;
}
```

**Understand:**

* When to use each function.
* Why the destination array must have enough space.
* Difference between `==` and `strcmp()` for strings.

# Session 2 – Aptitude (40 Minutes)

## Topic

### Simple Interest & Compound Interest

### Learn (15 Minutes)

### Simple Interest

```
SI = (P × R × T) / 100
```

Where:

* P = Principal
* R = Rate
* T = Time

### Compound Interest

```
Amount = P × (1 + R/100)^T

CI = Amount − Principal
```

---

### Practice (25 Minutes)

1. Find the simple interest on ₹5,000 at 8% for 3 years.
2. Find the amount on ₹10,000 at 10% for 2 years (compound interest).
3. A sum becomes ₹12,000 after 2 years at 10% simple interest. Find the principal.
4. Find the compound interest on ₹8,000 at 5% for 2 years.
5. Compare the SI and CI on ₹10,000 at 10% for 2 years.
