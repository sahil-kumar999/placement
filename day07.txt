# Session 1 – C Programming (100 Minutes)

## Topic

### Two-Dimensional (2D) Arrays

### Learn (30 Minutes)

Study these concepts:

* What is a 2D array?
* Declaration and initialization
* Row and column indexing
* Taking input using nested loops
* Displaying a matrix
* Matrix addition
* Matrix transpose (basic idea)

Example:

```c
#include<stdio.h>

int main()
{
    int a[2][3];
    int i, j;

    printf("Enter 6 elements:\n");

    for(i = 0; i < 2; i++)
    {
        for(j = 0; j < 3; j++)
        {
            scanf("%d", &a[i][j]);
        }
    }

    printf("Matrix:\n");

    for(i = 0; i < 2; i++)
    {
        for(j = 0; j < 3; j++)
        {
            printf("%d ", a[i][j]);
        }
        printf("\n");
    }

    return 0;
}
```

## Debugging Challenge (20 Minutes)

Find and fix the errors:

```c
#include<stdio.h>

int main()
{
    int a[3][3];
    int i, j;

    for(i = 0; i <= 3; i++)
    {
        for(j = 0; j <= 3; j++)
        {
            scanf("%d", &a[i][j]);
        }
    }

    return 0;
}
```

### Questions

* Why does this access memory outside the array?
* What should the loop conditions be?
* Rewrite the corrected program.

# Session 2 – Aptitude (40 Minutes)

## Topic

### Time & Work

### Learn (15 Minutes)

Important formulas:

```text
If A completes a work in N days,

One day's work = 1/N

Combined work:

(1/A) + (1/B)
```

Study:

* Individual work
* Combined work
* Efficiency

### Practice (25 Minutes)

1. A completes a job in **12 days**. How much work does A complete in one day?
2. B completes a job in **15 days**. What is B's one-day work?
3. A takes **10 days**, B takes **15 days**. In how many days will they complete the work together?
4. A and B together complete a task in **6 days**. If A alone takes **10 days**, how long will B take?
5. A can complete a job in **20 days**. After working for **5 days**, what fraction of the work remains?
